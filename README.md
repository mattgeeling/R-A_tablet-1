# R&A tablet interactive starter

Static scaffold for the R&A tablet build, based on:

- Adobe XD artboard names extracted from the share link
- `R&A.pdf` exported from XD
- supplied brand colors

## Files

- `index.html`: app shell and tablet stage
- `styles.css`: theme tokens and starter layout
- `src/screens.js`: XD artboard map
- `src/app.js`: simple screen router and placeholder content

## Notes

- The current starter does not include production image assets.
- The landscape sequence is modeled as one stacked composition plus three individual layer states.
- Replace placeholder panels/cards with final exported assets or content modules once available.
